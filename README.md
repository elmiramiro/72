# Модуль 17. Музыкальная коллекция — Задание 1

Файл `01_views_task1_music_collection.sql` содержит SQL-скрипт
с определениями представлений для БД «Музыкальная коллекция»
в соответствии с заданием 1 (Модуль 17, ч.5.1).

Представления:

1. `v_all_performers` — список исполнителей.
2. `v_songs_full_info` — полная информация о песнях.
3. `v_albums_info` — агрегированная информация об альбомах.
4. `v_most_popular_performer` — самый популярный исполнитель по числу дисков.
5. `v_top3_performers` — топ-3 исполнителей по числу дисков.
6. `v_longest_album` — самый долгий по длительности альбом.
